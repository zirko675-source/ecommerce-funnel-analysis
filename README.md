## Опис проєкту
GA4 E-commerce Funnel Analysis
Опис проєкту

Дашборд створений для аналізу поведінки користувачів в e-commerce проєкті на основі даних Google Analytics 4.

Метою аналізу є дослідження шляху користувача від початку сесії до покупки, виявлення етапів із найбільшим відтоком користувачів та оцінка конверсії в покупку.

Використані інструменти
SQL
Tableau Public
Google Analytics 4
Посилання на дашборд

https://public.tableau.com/app/profile/zirko.olena/viz/Pr1_17633061157620/Dashboard1

Бізнес-питання

У межах проєкту було досліджено:
загальну конверсію в покупку;
проходження користувачів через етапи воронки продажів;
точки найбільшого відтоку користувачів;
відмінності в конверсії залежно від типу пристрою;
відмінності в конверсії залежно від мови користувача;
динаміку конверсії в часі.

Ключові показники
Загальна кількість сесій: 356 626
Кількість покупок: 4 848
Конверсія в покупку: 1,37%

Аналіз воронки продажів

Воронка охоплює такі етапи:
Session Start
View Item
Add to Cart
Begin Checkout
Add Shipping Information
Add Payment Information
Purchase

Аналіз показав значне зменшення кількості користувачів між початком сесії та переглядом товарів, що свідчить про потенційні можливості для покращення залучення відвідувачів.

Аналіз за типом пристрою

У дашборді реалізовано порівняння конверсії між різними типами пристроїв:

Mobile — 1,41%

Desktop — 1,34%

Tablet — 1,32%

Найвищу конверсію демонструють користувачі мобільних пристроїв.

Аналіз у часі

Дашборд дозволяє відстежувати:
зміну кількості користувачів у часі;
динаміку конверсії в покупку;
можливі сезонні коливання показників.

Передбачено фільтрацію даних за:

країною;
типом пристрою;
мовою користувача;
періодом аналізу.
